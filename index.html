HP-42S RPN Scientific Calculator

The HP-42S is a programmable scientific calculator introduced by Hewlett-Packard in 1988. It was designed as a compact, powerful successor to the HP-41 series, offering software compatibility while lacking expandability and I/O features. It was marketed more as a modern replacement to the HP-15C.

Key Features

  • Reverse Polish Notation (RPN) entry
  • 2-line LCD dot-matrix display (22 characters, 131×16 pixels)
  • Over 350 built-in functions including matrix operations, complex numbers, equation solving, and numerical integration
  • Graphic output via programmable plotting
  • 64KB ROM, 7200 program steps, IR output to HP 82240A printer
  • Programmability via FOCAL-like keystroke programming

Emulators & Successors

Popular emulators include Free42 and Emu42 (for Android and desktop). The WP-34S project is a modern RPN scientific calculator based on the HP 30b hardware with extended functionality.
