HP-42S Scientific Calculator

The HP-42S is a powerful programmable scientific calculator introduced by Hewlett-Packard in 1988. A spiritual successor to the HP-41C series, it retained software compatibility but traded expandability for a sleek, compact form and a two-line dot-matrix display.

Highlights & Capabilities

  • Reverse Polish Notation (RPN) for efficient data entry
  • Two-line 131×16 pixel LCD with dot-matrix graphics
  • Support for matrices, complex numbers, equations, and integrals
  • 7200 program steps with structured memory
  • Infrared output to HP 82240A/B printers
  • Elegant FOCAL-style keystroke programming

Program Example: Area of a Circle

LBL "AREA"
INPUT R
X↑2
PI
×
RTN

This snippet prompts for a radius and returns the area using π·r².

Legacy & Modern Access

Though discontinued, the HP-42S has lived on via emulators like Free42 and hardware recreations like the DM42. Its precision and elegant design make it a favorite among engineers and calculator enthusiasts.
